5.1 - Positioning

Positioning elements using CSS can be very fine grained with the right selectors and attributes. Setting the position attribute of an element allows you to place it anywhere you like with greate precision.

position values
static
This is the default value for all browsers for any element
realtive
Setting the position to relative allows you to place an object relative to where it would be by default
fixed
Fixed positioning allows you to place an object relative to the "viewport" or the browser window. Any element with fixed positioning will not scroll with the rest of the page.
absolute
Absolute positioning positions elements relative their nearest ancestor or enclosing tag.

The next thing you'll need for precise positioning is the top and left attributes
